# Scripts

このディレクトリには自動化スクリプトを配置します。

## 構造

```
scripts/
├── setup.sh          # セットアップスクリプト
├── deploy.sh         # デプロイスクリプト
├── backup.sh         # バックアップスクリプト
└── cleanup.sh        # クリーンアップスクリプト
```

## 使用方法

スクリプトは実行可能にする必要があります:

```bash
chmod +x scripts/*.sh
./scripts/setup.sh
```

## 追加する内容の例

- 環境セットアップ自動化
- デプロイメント自動化
- バックアップ/リストア
- ログ収集
- ヘルスチェック
